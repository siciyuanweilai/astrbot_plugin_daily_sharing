## 📖 序言：打破“被动回复”的边界

**Daily Sharing** 是为 AstrBot 打造的核心主动交互插件。让Bot不再是一个只会“一问一答”的冰冷程序，而是一个会看时间、看天气、懂群聊气氛，甚至拥有自己“生活轨迹”的赛博生命。

无论是清晨带着阳光的一句早安语音、午后摸鱼时的一张热搜长图、深夜网抑云时的一段走心视频，还是自动分享到QQ空间的日常说说……Bot都能完美胜任。

## 🌟 核心特性 

### 1. 🔍 双引擎防幻觉：Tavily 联网增强 
*   **拒绝热搜标题党**：自动接入全网检索能力。分享新闻时检索“事件最新进展与始末”，带脑子吃瓜，不再仅凭标题进行空洞脑补。
*   **百科+网页并发校验**：在进行【知识】与【推荐】时，并发调用“百度百科 + Tavily 搜索”。拥有扎实的数据支撑，从根本上解决 LLM 胡编乱造的幻觉问题。

### 2. 🌐 跨域社交：QQ空间与独立早报联动 
*   **说说自动机**：无需人工干预，Bot会根据一天中的不同时段（早晨问候/深夜emo），自动以“第一人称”口吻生成配图说说并发布到QQ空间。
*   **早报速递员**：支持将【每天 60s 读懂世界】和【AI 资讯快报】作为独立定时任务，自动推送到群聊、私聊并分享至QQ空间。

### 3. 🧠满级情商：认知防呆与动态记忆
*   **KY 规避 (读空气)**：群聊态势感知。当群友正在热火朝天地讨论时，Bot会保持安静，绝不打断施法。
*   **记忆回写 (拒绝断片)**：Bot发送的配图都会被**写回AstrBot的对话历史**。你问：“你刚才发的照片是在哪拍的？”，它能对答如流。
*   **第二人称防呆**：深度优化底层提示词，彻底解决Bot在私聊时把用户当“第三人称”称呼的语病，交互更像真人伴侣。

### 4. 🎨 视觉与影像：形象参考与视频生成
*   **形象一致性**：联动新版**Gitee**生图插件，支持**形象参考图**。Bot发的生活照终于可以拥有稳定的“脸”和气质了！
*   **图文转视频**：一键将生成的静态生活照转换为**高清动态视频**。清晨的早安，是真的在对你挥手微笑。
    **构图顾问**：内置**Agent**智能判断——分享心情时“画人”，分享新闻时“画景”。
*   **环境光影**：自动根据物理时间（清晨/黄昏/深夜）调整画面的光影，杜绝 "半夜窗外是大太阳" 的违和感。

### 5. 📰 分享内容：全网热搜与自定义知识库
*   **众多热搜源**：微博、知乎、B站、抖音、头条、百度、腾讯、小红书、夸克、36氪、51CTO、A站、爱范儿、网易、新浪、澎湃、第一财经。
*   **时段偏好算法**：早晨看知乎/小红书，傍晚刷 B 站/微博，深夜看抖音，完美契合人类作息。
*   **专属内容库**：支持在配置项中完全自定义【知识】和【推荐】库的话题方向。

### 6. 🛡️ 极高可用性：双重降级机制
*   **LLM 自动降级**：指定的大模型 API 挂了？超时了？触发敏感词了？插件会自动无缝切换到系统默认的备用模型，确保分享从不缺席。
*   **新闻源熔断**：主新闻源获取失败时，自动在备选池中切换；全网瘫痪时，动用 LLM 内部知识库兜底生成。

### 7. ⏰ 工业级调度：灵活的调度策略与断电防丢
*   **时段序列**：不同时间段执行不同的逻辑（早晨发问候，下午发推荐，晚上发心情）。
*   **Cron 支持**：内置预设（`morning`, `twice`）或完全自定义 Cron 表达式。
*   **多时段随机触发**：支持配置多个时间段（如 08:00-10:00），系统每天自动随机抽选时间点，像真人一样不规律作息。
*   **任务持久化恢复**：所有的“定时随机延迟”均写入 SQLite 数据库。即使 Bot 重启、断网或崩溃，也能在启动时精准恢复未执行的分享任务，**彻底告别重启导致的漏发或连发**。

## 🧩 推荐插件生态

本插件可以独立运行（纯文字模式），但强烈建议安装以下插件以获得完整体验：

| 插件名称 | 必需性 | 作用描述 | 缺失影响 |
| :--- | :--- | :--- | :--- |
| **[<astrbot_plugin_qzone](https://github.com/Zhalslar/astrbot_plugin_qzone)** | ⭐⭐⭐⭐⭐ | **QQ空间基建** | 无法将日常分享、早报长图自动发布到空间说说 |
| **[astrbot_plugin_life_scheduler](https://github.com/muyouzhi6/astrbot_plugin_life_scheduler)** | ⭐⭐⭐⭐⭐ | **虚拟生活模拟** | 失去天气、日程感知，无法根据时间调整穿搭和所处场景 |
| **[astrbot_plugin_gitee_aiimg](https://github.com/muyouzhi6/astrbot_plugin_gitee_aiimg)** | ⭐⭐⭐⭐ | **图生图/文生视频** | 无法发送生活配图，无法生成视频消息 |
| **[astrbot_plugin_memos_integrator](https://github.com/zz6zz666/astrbot_plugin_memos_integrator)** | ⭐⭐⭐⭐ | **长效记忆存储** | 无法引用长线的历史话题，容易“车轱辘话” |
| **[astrbot_plugin_tts_emotion_router](https://github.com/muyouzhi6/astrbot_plugin_tts_emotion_router)**| ⭐⭐⭐⭐ | **情感语音合成** | 无法根据文案情绪发送对应的语音条 |

## 💬 沉浸式自然语言交互

得益于**AstrBot**的工具函数调用能力，**无需记忆繁琐指令**，直接和Bot聊天即可触发分享！

> **💡 触发秘籍**：你可以在任何请求后面加上后缀要求，例如**“要视频哦”**、**“发一段语音”** 或 **“发条说说”**。

| 你的意图 | 示例说辞 (自然语言) | Bot的实际动作 |
| :--- | :--- | :--- |
| **发空间说说** | “今天天气不错，**发个空间说说吧**” | 生成第一人称心情独白，并附带生活照发布至 QQ 空间。 |
| **看动图/视频** | “早上好，给我**发个自拍视频**看看” | 强制触发画图+视频生成，发送一段会动的问候视频。 |
| **看独立早报** | “给我来一份 **60s 新闻**” / “看看 **AI资讯**” | 直接获取当天的《60s 读懂世界》长图并发送。 |
| **吃瓜看热搜** | “看看**微博热搜**的图片版” | 精准获取微博实时热搜长图。 |
| **听碎碎念** | “你现在心情怎么样？**发条语音**” | 结合它此时的“虚拟日程”生成一段音频发送给你。 |

## 🎮 控制台指令 (Admin Only)

| 指令格式 | 参数示例 | 功能说明 |
| :--- | :--- | :--- |
| `/分享` | `[类型]` | 手动触发分享（默认只发给当前窗口）。 |
| `/分享` | `[类型] 广播` | 向配置中的所有群聊和私聊发送。 |
| `/分享` | `[类型] 空间` | 单独生成文案并一键分享到QQ空间。 |
| `/分享` | `新闻 [源]` | 获取指定源的热搜（文字版） |
| `/分享` | `新闻 [源] 图片` | 获取指定源的热搜长图。 |
| `/分享` | `60s` / `ai` | 立即手动发送【每天60s读懂世界】或【AI资讯快报】。 |
| `/分享` | `早报空间 开启/关闭` | 切换是否将定时早报自动分享到QQ空间。 |
| `/分享` | `开启` / `关闭` | 启停所有定时自动分享任务。 |
| `/分享` | `状态` | 查看运行状态、历史记录、序列索引 |
| `/分享` | `查看序列` | 查看当前时间段会轮换发什么内容 |
| `/分享` | `指定序列 [序号]` | 手动跳到序列中的某一步 (支持加后缀 `空间`) |
| `/分享` | `重置序列` | 重置轮换顺序，回到列表开头 |

## ⚙️ 进阶配置指南 (Config)

请在 AstrBot WebUI 的 **插件配置** 面板中进行调整。

<details>
<summary><b>点击展开：核心配置项说明</b></summary>

1. **🎯 千群千面推送配置**
   - 推送目标支持三段式高级语法：`群号:独立定时:分享类型`。
   - 例如填写 `123456:0 7 * * *:news,mood`，意味着该群脱离全局设定，拥有独立的早7点闹钟，且只轮换发送新闻和心情。
2. **📚 内容库自定义**
   - 自定义知识和推荐的话题池。格式为 `大类: 标签1, 标签2`。（例如：`游戏: 独立神作, 治愈解谜`）。
3. **📅 早报设置**
   - 独立配置每天 `08:00` 发送的长图早报。支持一键开启分享到 QQ 空间。
4. **🌟 QQ空间专属设置**
   - 拥有完全独立于群聊/私聊的 Cron 定时器（如 `0 20 * * *`）和发布序列。
   - 自动屏蔽“大家好”等社交词汇，强制输出第一人称“日记体”。
5. **🎨 配图与视频**
   - **强烈建议开启**，配合 Gitee 插件实现人设外貌的绝对稳定。
   - 视频生成开关，仅建议为 `greeting` 和 `mood` 开启，避免过度消耗API。
6. **🔍 联网检索**
   - 默认开启。结合 AstrBot 内置的 Tavily 搜索功能，让 Bot 的热搜点评、知识分享更具真实性和深度。   
7. **🤖 模型与人设**
   - 可单独指定负责此插件的 LLM 模型。如果该模型崩溃，插件会自动降级寻找系统中的第一个可用模型。

</details>

## 🌰 沉浸式场景展示

**▶ 场景 A：早间生活记录 (同步 QQ 空间)**
> **系统触发**：早上 8:00 定时任务。
> **Bot 内部逻辑**：检测到 Gitee 参考图启用 ➡ 提取当前天气和穿搭 ➡ 过滤社交语句 ➡ 调用 Qzone 插件。
> **最终呈现 (QQ空间说说)**：“清晨的阳光透过窗帘洒进来，手里的咖啡还在冒着热气。换上了最喜欢的米色毛衣，感觉今天会是充满希望的一天呀~ ☕️ ✨ *(配图：一张构图精美、完美符合人设的咖啡厅半身自拍照)*”

**▶ 场景 B：防打扰的倾听者 (群聊)**
> **系统触发**：晚上 20:00 定时新闻推送。
> **Bot 内部逻辑**：拉取最近 80 条消息 ➡ 计算群聊热度 ➡ 发现群友正在激烈讨论《黑神话》打法。
> **日志打印**：`[DailySharing] 监测到群聊热度过高，策略为 cautious，跳过本次主动分享。` (Bot 保持静默，深藏功与名)
